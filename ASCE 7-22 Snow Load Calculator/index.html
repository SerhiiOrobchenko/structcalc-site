This module calculates the balanced roof snow load per ASCE/SEI 7-22, Chapter 7 ("Snow Loads"): the exposure factor Ce (Table 7.3-1), thermal factor Ct (Tables 7.3-2/7.3-3), flat roof snow load pf (Eq. 7.3-1), slope factor Cs (Figure 7.4-1 / Commentary C7.4), sloped roof snow load ps (Eq. 7.4-1), the minimum snow load pm for low-slope roofs (Sec. 7.3.3, Table 7.3-4), the rain-on-snow surcharge (Sec. 7.10), and the sliding snow load onto a lower roof (Sec. 7.9). Every value is computed explicitly with its clause/equation/table reference so it can be checked against your own copy of the Standard.

Ground snow load, pg: ASCE 7-22 has no separate snow importance factor — pg itself is Risk-Category-specific, taken from Figures 7.2-1A–D (Table 7.2-1 for Alaska). Look up pg for your site and Risk Category using the official ASCE 7 Hazard Tool (ascehazardtool.org) and enter it below.

Items not yet covered by this module (drift loads on lower roofs, parapets/projections, unbalanced loads, partial loading, curved/multi-roof slope factors, ponding, etc.) are listed in the Roadmap in the footer below.

Sources, Assumptions & Roadmap

Where these formulas come from

  • ASCE/SEI 7-22, Sec. 7.2 — Ground snow load, pg, is Risk-Category-specific (no separate snow importance factor in 7-22). Determined from Figures 7.2-1A–D, or Table 7.2-1 for Alaska. Use the official ASCE 7 Hazard Tool for your site coordinates.
  • Table 7.3-1 — Exposure Factor, Ce: Surface Roughness B/C/D × Fully/Partially Exposed/Sheltered (B: 0.9/1.0/1.2; C: 0.9/1.0/1.1; D: 0.8/0.9/1.0), Above tree line and Alaska (no trees within 2 mi): 0.7/0.8/N/A.
  • Table 7.3-2 — Thermal Factor, Ct: "All structures except as indicated" → Table 7.3-3; Unheated/open-air/cold ventilated roofs meeting energy code → Ct=1.2; Freezer buildings → Ct=1.3; Continuously heated greenhouses (R<2.0) → Ct=0.85.
  • Table 7.3-3 — Thermal Factor, Ct, for heated structures with unventilated roofs, as a function of Rroof (20/30/40/≥50 h·ft²·°F/Btu) and pg (≤10…≥70 psf), ranging 1.00–1.20; linear interpolation per Note (a); Rroof > 50 → Ct=1.2 per Footnote (b). This calculator performs bilinear interpolation on the full table.
  • Eq. 7.3-1 — Flat roof snow load: pf = 0.7·Ce·Ct·pg.
  • Sec. 7.4.1, Fig. 7.4-1, Commentary C7.4 — Slope factor Cs, reproduced mathematically in Commentary C7.4 (ASCE/SEI 7-22, p.592): for Ct<1.1, single "all other surfaces" curve (1.0 for slope≤30°, linear to 0 at 70°, denominator 40°); for 1.1≤Ct<1.2, slippery curve (1.0 to 10°, denom 60°) and "all other" curve (1.0 to 37.5°, denom 32.5°); for Ct≥1.2, slippery curve (1.0 to 15°, denom 55°) and "all other" curve (1.0 to 45°, denom 25°). Per C7.4, Figure 7.4-1(a) has no slippery-surface curve (ice damming assumed) when Ct<1.1.
  • Eq. 7.4-1 — Sloped (balanced) roof snow load: ps = Cs·pf.
  • Sec. 7.3.3, Table 7.3-4 — Minimum snow load for low-slope roofs, pm: applies only to monoslope/hip/gable roofs with slope <15°, or curved roofs with eave-to-crown angle <10°. pm = pg if pg≤pm,max, else pm,max (i.e., pm=min(pg,pm,max)). pm,max by Risk Category: I=25 psf (1.20 kPa), II=30 psf (1.44 kPa), III=35 psf (1.68 kPa), IV=40 psf (1.92 kPa). pm is a separate uniform load case, not combined with drift, sliding, unbalanced, or partial loads.
  • Sec. 7.9 — Sliding snow: applies for slippery upper roofs with slope >1/4 on 12 (1.19°), and for other (non-slippery) upper roofs with slope >2 on 12 (9.46°). Total sliding load per unit length of eave = 0.4·pf·W, where W is the horizontal eave-to-ridge distance, distributed over 15 ft (4.6 m) of the lower roof (reduced proportionally if the lower roof is narrower).
  • Sec. 7.10 — Rain-on-snow surcharge: for sites where 0<pg≤pm,max (Table 7.3-4) and roof slope (degrees) < W/50 (W in ft; W/15.2 with W in m), add an 8 psf (0.38 kPa) surcharge to the sloped-roof balanced load ps only — not combined with drift, sliding, unbalanced, minimum, or partial loads.

Site Location & Seismic Design Parameters panel

  • Open-Meteo Geocoding API (geocoding-api.open-meteo.com/v1/search) — converts the entered address to latitude/longitude (and elevation). Free, no API key, CORS-enabled for direct browser use; results are derived from OpenStreetMap place data.
  • Leaflet.js + OpenStreetMap tiles — renders the located point on a map. No API key required.
  • USGS Design Maps web service (earthquake.usgs.gov/ws/designmaps/asce7-22.json) — for the given latitude/longitude, Risk Category, and Site Class, returns SS, S1, SMS, SM1, SDS, SD1, PGAM, TL, TS, T0, and Seismic Design Category, per ASCE/SEI 7-22 Sec. 11.4 (site coefficients Fa/Fv and the SDS=(2/3)SMS, SD1=(2/3)SM1 relations) and Tables 11.6-1/11.6-2 (Seismic Design Category). This is the same USGS service referenced by the ASCE Hazard Tool and most third-party seismic design software.
  • Site Class, Ch. 20 — ASCE/SEI 7-22 added intermediate Site Classes BC, CD, DE to Table 20.2-1 alongside A–F. Site Class must be determined by a geotechnical investigation per Ch. 20; "Default" applies the Sec. 11.4.3 default when site soil properties are not known in sufficient detail. Site Class F (requiring a site response analysis, Sec. 11.4.8) is not provided by the USGS service.
  • This panel supplies seismic design parameters only. It does not perform a seismic base shear or lateral force analysis (ASCE 7-22 Ch. 12) — that remains a future module. If the USGS request fails (e.g., a network/CORS restriction in your browser), the parameters can be looked up manually at earthquake.usgs.gov/ws/designmaps or via the ASCE 7 Hazard Tool.
About Rroof and Table 7.3-3: ASCE 7-22 changed Table 7.3-2 so that ordinary heated structures ("all structures except as indicated") no longer get a flat Ct=1.0 — instead, Ct is read from Table 7.3-3 as a function of the roof’s thermal resistance Rroof and the ground snow load pg, ranging from 1.00 (well-insulated, low pg) to 1.20 (poorly insulated, high pg, or Rroof>50). This calculator implements the full Table 7.3-3 grid with bilinear interpolation, exactly as printed.

  • Wind loads (ASCE 7-22 Ch. 26–30) and seismic loads (ASCE 7-22 Ch. 11–12, with USGS Design Maps lookup) — separate modules.
